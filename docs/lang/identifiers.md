# Identifiers

Identifier is a combination of small & big letters, digits and '_' sign.
Every identifier must starts from letter.

## Type identifier
*Type identifier* must starts from **uppercese** letter

#### Example

```swift
type MyInt Int64
type PointerToMyInt *MyInt
```

## Value identifier
*Value identifier* must starts from **lowercase** letter

#### Example
```swift
const myConst = 10

var myVar: MyInt

func main () -> Int32 {
    return 0
}

```
