# Access Modfiers

## Private (default)

#### Example
```
// you can leave 'private' keyword
private const secretConst = 42

// access modifier by default is 'private'
// so this function is private
func secretFunc () -> Unit {
	// do something secret
}
```

## Public

#### Example
```
public const moduleName = "Main"

func main () -> Unit {
	// do something
}
```
