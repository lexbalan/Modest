# Function Definition

Function definition creates new *function*.

#### Common form
```
func <#identifier#> <#func_type_expression#> {
	// function block
}
```

#### Examples

```swift

func sum_i32 (a: Int32, b: Int32) -> Int32 {
	return a + b
}

func main () -> Int32 {
	printf("Hello World!\n")
    return 0
}

```
